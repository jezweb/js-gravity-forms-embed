Implementation Notes

  • No dependencies required: All themes use pure CSS without requiring Bootstrap, Tailwind, or other frameworks
  • Responsive: All themes are mobile-friendly and adapt to different screen sizes
  • Customizable: Use custom CSS to further modify any theme
  • Performance: Themes add minimal overhead to form loading
  • Accessibility: All themes maintain proper contrast ratios and keyboard navigation

Advanced Usage

// With API key
<div data-gf-form="1" data-gf-theme="dark" data-gf-api-key="your-api-key"></div>

// Multiple forms with different themes
<div data-gf-form="1" data-gf-theme="bootstrap"></div>
<div data-gf-form="2" data-gf-theme="tailwind"></div>

// With custom container styling
<div style="max-width: 600px; margin: 0 auto;">
    <div data-gf-form="1" data-gf-theme="glass"></div>
</div>
