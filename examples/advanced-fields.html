üîß Implementation

To use these advanced features, simply embed your form as usual:

<!-- Basic embed -->
<div data-gf-form="1"></div>
<script src="https://your-site.com/gf-js-embed/v1/embed.js"></script>

The SDK automatically detects and initializes advanced field types!

üéÆ JavaScript Events

Listen for form events to customize behavior:

// Form ready event
document.addEventListener('gfEmbedFormReady', function(e) {
    console.log('Form loaded:', e.detail.formId);
});

// Submission success
document.addEventListener('gfEmbedSubmitSuccess', function(e) {
    console.log('Form submitted:', e.detail.entryId);
});

// Validation errors
document.addEventListener('gfEmbedValidationError', function(e) {
    console.log('Validation errors:', e.detail.errors);
});
